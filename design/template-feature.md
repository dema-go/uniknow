# 案例模版功能设计文档

## 1. 功能概述

在编辑器页面右侧增加悬浮工具栏，提供案例模版、搜索、评分、润色等辅助功能，提高案例创建效率。

## 2. 功能列表

| 功能 | 优先级 | 说明 |
|------|--------|------|
| 案例模版 | P0 | 选择预置模版，自动填充到编辑器 |
| 搜索 | P1 | 搜索已有案例（待实现） |
| 案例评分 | P1 | AI 评估案例质量（待实现） |
| 案例润色 | P1 | AI 优化案例内容（待实现） |

## 4. 内置模版

### 4.1 FAQ标准模版

```markdown
## 问题描述
[在此描述问题]

## 解决方案
1. 步骤一
2. 步骤二
3. 步骤三

## 注意事项
- 注意点1
- 注意点2

## 相关链接
- [链接名称](URL)
```

### 4.2 流程指引模版

```markdown
# 流程名称

## 适用范围
[描述适用对象]

## 前置条件
- 条件1
- 条件2

## 操作步骤

### 步骤1: [步骤名称]
[详细说明]

### 步骤2: [步骤名称]
[详细说明]

## 常见问题
| 问题 | 解决方案 |
|------|----------|
| 问题1 | 方案1 |
```

### 4.3 公告通知模版

```markdown
# [公告标题]

**发布时间**: YYYY-MM-DD
**生效时间**: YYYY-MM-DD

## 摘要
[简要说明公告内容]

## 详细内容
[详细内容描述]

## 影响范围
- 影响对象1
- 影响对象2

## 注意事项
[需要用户注意的事项]
```

## 5. 技术实现

### 5.1 前端组件结构

```
frontend/src/
├── components/
│   └── editor/
│       ├── EditorSidebar.vue    # 悬浮侧边栏
│       ├── SidebarButton.vue    # 侧边栏按钮
│       └── TemplateDialog.vue   # 模版选择弹窗
├── data/
│   └── templates.js             # 内置模版数据
└── views/
    └── cases/
        └── create.vue           # 集成侧边栏
```

### 5.2 后端 API（预留）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | /api/v1/templates | 获取模版列表 |
| GET | /api/v1/templates/{id} | 获取模版详情 |
| POST | /api/v1/templates | 创建自定义模版 |
| PUT | /api/v1/templates/{id} | 更新模版 |
| DELETE | /api/v1/templates/{id} | 删除模版 |
